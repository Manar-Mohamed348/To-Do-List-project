## Key Features  
- **One-click completion** - Tap tasks to toggle gray/strikethrough  
- **Clean interface** - Distraction-free design with calming colors  
- **Dark/Light modes** - Eye-friendly theme switching  
- **Date awareness** - Always shows current day/date  
- **Quick actions** - Add, delete, or clear tasks instantly  

## 🎮 How to Use  
1. **Add task**: Type and press "Add Task" Button 
2. **Complete**: Click task → turns gray with ~~strikethrough~~  
3. **Delete**: Click ❌ next to task  
4. **Toggle theme**: Click moon/sun icon
